# DevProduct Manager
<img width="2559" height="1291" alt="image" src="https://github.com/user-attachments/assets/68918176-671d-4ba0-88c8-69006e1e44cb" />


Chrome extension for managing Roblox developer products. View, edit, create, and bulk manage DevProducts with comfort.

## Features

- View all developer products for any Roblox experience
- Inline editing of product names and prices
- Create new products
- Bulk create multiple products at once
- Export products to JSON
- Works from the Roblox experience page or as a standalone tab

## Setup

```bash
npm install
```

## Build

```bash
npm run build
```

Load the `dist` folder as an unpacked extension in Chrome.

## Usage

1. Navigate to any Roblox experience page
2. Click the extension icon

## Tech

- React
- TypeScript
- Vite
